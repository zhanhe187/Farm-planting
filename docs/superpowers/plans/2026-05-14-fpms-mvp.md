# FPMS MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a runnable Java/Spring Boot MVP of the farm planting management system from the planning document.

**Architecture:** Create a Spring Boot 2.7 Thymeleaf application with H2 as the default development database and SQL Server scripts for production-shaped features. Keep domain behavior in testable services: status machine, inventory consumption, safety interval enforcement, trace view generation, AI fallback, and mobile bind/recognition.

**Tech Stack:** Java 8 target, Spring Boot 2.7.18, Maven, Thymeleaf, JdbcTemplate, H2, SQL Server driver, MyBatis-Plus dependency, Caffeine, zxing, WebFlux, ECharts/LayUI-style static UI.
