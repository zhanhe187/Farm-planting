# FPMS 落地部署与运行说明

## SQL Server 初始化

当前工程默认使用 SQL Server，项目数据全部保存在 `daihaojie_xm` 数据库中。

连接信息：

- 服务地址：`localhost:1433`
- 数据库：`daihaojie_xm`
- 用户名：`sa`
- 密码：`123456`

首次运行前执行：

```powershell
cd D:\javacode\fpms-mvp
sqlcmd -S localhost -U sa -P 123456 -C -b -f 65001 -i src\main\resources\db\init-daihaojie-xm.sql
```

脚本会在数据库不存在时创建 `daihaojie_xm`，并完成建表、触发器、视图和初始化数据写入。

## 本地快速启动

```powershell
cd D:\javacode\fpms-mvp
mvn spring-boot:run
```

访问地址：

- 后台入口：`http://localhost:8080`

## 初始化账号

本地初始化数据提供以下账号，初始密码均为 `123456`。

| 账号 | 角色 | 用途 |
| --- | --- | --- |
| `admin` | SUPER_ADMIN | 系统配置、AI 端点、全量数据 |
| `owner` | FARM_OWNER | 农场主经营视角 |
| `agri` | AGRI_TECH | 农技员视角 |
| `warehouse` | WAREHOUSE | 库存管理 |
| `worker` | FIELD_WORKER | 田间作业 |
| `customer` | CUSTOMER | 经销客户 |

## 当前本地回退能力

以下能力保留了真实接口边界，但默认使用本地回退实现，便于先落地业务闭环：

- AI Chat：`/ai/chat` 与 `/ai/stream` 返回本地农事建议，后续可接 OpenAI 兼容端点。
- AI Vision：手机端识作物返回结构化结果，后续可接视觉模型。
- 手机扫码：本地显示一次性绑定链接；部署时可用 zxing 生成二维码图片。

## 生产增强建议

- 使用 Sa-Token 或 Spring Security 接管完整鉴权、权限注解、会话失效策略。
- 使用 Flyway 管理 SQL Server 脚本版本。
- 将 AI API Key 使用 AES-GCM 加密落库，主密钥放入环境变量或 KMS。
- 文件上传切换到独立对象存储或受控本地目录。
- 增加操作日志、异常审计、库存并发集成测试和公开溯源限流。
